RUN COMMANDS
============

1) Open PowerShell inside this folder:
   cd "C:\Users\jktyr\Downloads\rag-app-main-updated-vector-db"

2) Install packages:
   npm install --legacy-peer-deps

3) Add your .env values. Important:
   GROQ_API_KEY=your_key
   VECTORSTORE_MAX_SIZE_MB=1024

4) Run:
   npm start
